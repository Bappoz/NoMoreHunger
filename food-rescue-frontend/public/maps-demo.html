⚙️ Como Configurar a Integração

⚠️ Importante: Para usar todas as funcionalidades, você precisa de uma chave da API do Google Maps.
  1. Acesse o Google Cloud Console
  2. Crie um projeto ou selecione um existente
  3. Ative as APIs necessárias:
    • Maps JavaScript API
    • Geocoding API
    • Places API (opcional)
  4. Crie uma chave de API em "Credentials"
  5. Configure restrições de segurança (HTTP referrers)
  6. Adicione a chave no arquivo .env:
    VITE_GOOGLE_MAPS_API_KEY=sua_chave_aqui
  7. Reinicie o servidor de desenvolvimento
✅ Modo Fallback: O sistema funciona mesmo sem a API key, oferecendo funcionalidades básicas de geolocalização e interface simulada.
